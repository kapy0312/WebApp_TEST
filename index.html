安裝 WEB APP 教學

手機-Android系統

1.開啟Chrome瀏覽器，載入遊戲連結。
2.進入主畫面之後，按下右上方的「...」按鈕，
   並點選「加到主畫面」。
3.點選「新增」來加入主畫面。
4.完成安裝。

手機-iOS系統

1.開啟Safari瀏覽器，載入遊戲連結。
2.進入主畫面後，點擊底下的分享。
3.點選「加入主畫面」。
4.點選「新增」來加入主畫面。
5.完成安裝。

PC-Windows系統

1.開啟Microsoft Edge瀏覽器，
   載入遊戲連結。
2.進入主畫面之後，按下右上方的「...」按鈕。
3.游標移動至「應用程式」，
   點選「將此網站安裝為應用程式」。
4.加選桌面捷徑後，點選「允許」。
5.完成安裝。

遊戲應用技術

PWA（漸進式網路應用程式）

PWA 指的是「Progressive Web App」，是一種結合網頁和應用程式功能的現代網頁技術。 PWA 的目標是提供優質的用戶體驗，使網站在行動設備上能夠像原生應用程式一樣運行， 同時具備網頁的優勢，如可訪問性、即時更新和跨平台支援。

RPG Maker MZ

RPG Maker MZ 是一款由 KADOKAWA 開發的遊戲開發工具，用於創建角色扮演遊戲(RPG)。 它是 RPG Maker 系列的最新版本，提供了一個強大且簡單易用的平台，讓開發者可以創建自己的 RPG 遊戲， 包括角色、地圖、劇情、戰鬥系統等。

Read More
GitHub Pages

GitHub Pages 是 GitHub 提供的一個免費的靜態網站托管服務，它允許用戶將自己的代碼庫 （Repositories）轉變為網站，並將這些網站在互聯網上進行訪問。 這是一個方便的方式來展示你的項目、個人網站、文檔等。

ChatGPT

ChatGPT 是 OpenAI 開發的一種語言模型，基於 GPT-3.5 架構，旨在進行自然語言處理任務。 它被訓練過大量的文本數據，能夠理解和生成人類語言，並能夠進行對話、回答問題、提供建議、 執行文本生成任務等。
